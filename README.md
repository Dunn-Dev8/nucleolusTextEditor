Clone this github repo
`git clone https://github.com/Dunn-Dev8/nucleolusTextEditor`

`cd /nucleolusTextEditor/Installer`

Make sure the installer.sh is executable
`chmod +x installer.sh`

Run installer.sh
`./installer.sh`
